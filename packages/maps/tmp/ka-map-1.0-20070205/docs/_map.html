Constructor Detail

_map

_map(o)
    _map internal class used to store map objects coming from the init script szName - string, the layer name (or group name, in this case ;)) szTitle - string, the human-readable title of the map nCurrentScale - integer, the current scale as an index into aszScales; aszScales - array, an array of scale values for zooming. The first scale is assumed to be the default scale of the map aszLayers - array, an array of layer names and statuses. The array is indexed by the layer name and the value is true or false for the status. ***************************************************************************
