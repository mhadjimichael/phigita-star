ka-Map API Documentation

This ka-Map API documentation describes how to use ka-Map to create your own applications, and how it can be extended to add new functionality and tools. This API is a work in progress, so it may be incomplete or even inaccurate in some cases. However, we are very keen to make it as complete, accurate and useful as possible, so please join the ka-Map users mailing list and provide us with any feedback you can.
